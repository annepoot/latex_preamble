
Significant conflicts:
\mX -> observation locations, sample matrix
\nu -> wave number, poisson's ratio
\phi -> eigenfunction, test function
\opG -> Green's function, observation operator

Acceptable conflicts:
d -> dimensionality, hole size, misspecification
m -> mean function, trial function count
w -> arbitrary function, prescribed displacement
x -> x, hole location
y -> hole location, observation

\ve -> bfemobservation noise, noise
\vf -> source field, source vector
\vu -> displacement field, solution vector

J -> energy, quantity of interest
H -> beam height, Hilbert space
L -> beam length, loss function, L-space

\eta -> relative interpolation error, hyperparameter
\theta -> parameter, log diffusion

\valpha -> perturbation, perturbations
\vsigma -> stress field, std vector
